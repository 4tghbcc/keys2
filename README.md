# Стилизованный Счетчик

Красивый и функциональный счетчик с градиентным дизайном и интерактивными элементами.

## Функциональность

- ✅ Увеличение/уменьшение значения кнопками  "+" и "-"
- ✅ Изменение цвета фона в зависимости от значения:
  - 🟡 Желтый - положительные значения
  - 🟢 Зеленый - отрицательные значения  
  - 🔴 Красный - ноль
- ✅ Блокировка кнопок при достижении пределов:
  - Кнопка "+" блокируется при значении 10
  - Кнопка "-" блокируется при значении -10
- ✅ Сообщение при достижении экстремальных значений
- ✅ Управление с клавиатуры (клавиши "+" и "-")
- ✅ Адаптивный дизайн для мобильных устройств
- ✅ Плавные анимации и переходы

## Технологии

- HTML5
- CSS3 (Grid, Flexbox, Gradients, Transitions)
- Vanilla JavaScript (ES6+)
- CSS Transitions & Animations
